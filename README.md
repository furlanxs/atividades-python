# Atividades de Python - Raciocínio Algorítmico

Este repositório foi criado para armazenar as atividades práticas desenvolvidas durante as aulas de Raciocínio Algorítmico do curso de Engenharia de Software da PUCPR.

## Conteúdo
* **Manipulação de Matrizes:** Exercícios práticos utilizando a linguagem Python para operações com matrizes.
* **Controle de Versão:** Demonstração do uso de Git e GitHub para gerenciamento de código.

## Tecnologias Utilizadas
* Python 3
* Git / GitHub
